** Description:
	* This file contains the instructions on how to compile and run the program
	* author: Kevin Jang (kj460)
	* last_updated: 11/28/18

** Files
	* Below is the list of files contained in this zip file
		1. myfingerprinter.py - python code that contains functions necessary to run the program
		2. Seventy training files ending with '.pcap'
			- Ten training files per site
		3. canvas_test.pcap - sample test file provided on the course website
	* In total this zip file contains 'seventy-three' files

** How to run
	* Below is the sample command to run the program:
		python myfingerprinter.py canvas_test.pcap
	* 'myfingerprinter.py' is the driver python file for this program
	* 'canvas_test.pcap' is the test pcap file
